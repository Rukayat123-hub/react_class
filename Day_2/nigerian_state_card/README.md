1. Create `StateCard.jsx`

3. Display all information in a styled card
4. Use different background colors for different regions:
   - North: `#8B4513`
   - South: `#008751`
   - East: `#FFD700`
   - West: `#4169E1`
5. Create 4 state cards in `App.jsx`
