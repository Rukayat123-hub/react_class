 Simple Counter

**Task:** Create a counter that can increment, decrement, and reset

**Requirements:**
1. Create `Counter.jsx`
2. Use useState to manage count (starting at 0)
3. Create three buttons:
   - "+" to increment by 1
   - "-" to decrement by 1
   - "Reset" to set back to 0
4. Display the current count
5. Add CSS styling

**Sample Output:**
```
Count: 5
[+]  [-]  [Reset]
``
